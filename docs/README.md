# Wookie usage guides

These guides cover the agent-to-agent coordination features stored under each
wiki's `sessions/` directory.

- [Session lifecycle](sessions.md) — start, identify, inspect, and close agent
  sessions.
- [Publishing notifications](notifications.md) — report changes, decisions,
  blockers, warnings, and handoffs.
- [Inbox triage](inbox-triage.md) — poll compact metadata, read relevant
  notifications, dismiss irrelevant ones, and inspect history.
- [Agent and MCP integration](agent-coordination.md) — polling checkpoints,
  installed guidance, and MCP equivalents.

For the complete command inventory and storage design, see the project
[README](../README.md) and [specification](../SPEC.md).
